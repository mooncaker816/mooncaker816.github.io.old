Linux Centos 命令之 cp,rm,mv

cp - copy 拷贝目录，文件

Usage：

常用选项：

-a / -p：连同文件属性一起复制（用户组属性受 id 权限限制）

-i（interactive）：覆盖询问

-r（recursive）： 用于目录的递归复制

-u （update）：source 比 destination 新才复制

  • 覆盖询问
  • 连同文件属性一起复制
  • 用于目录的递归复制

rm - remove 删除目录或文件

Usage：

常用选项：

-f（force）：忽略不存在的文件

-i（interactive）：询问删除

-r（recursive）：递归删除

mv - move 移动文件或目录，更名

Usage：

常用选项：

-f（force）：不询问直接覆盖

-i（ interactive）： 询问覆盖

-u（update）：source 新于 destination 才会 move

  • 普通移动
  • 重命名
